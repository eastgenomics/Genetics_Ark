Primer Database

Using the main page

This is the main view for the primers, they are all displayed here. From top to bottom:

  • Submit buttons
  • Filter using various parameters (stackable)
  • Change primer status (you can do it in bulk by checking them in the table)
  • Recalculating coverage will display new coverages for the pair of primers selected.
    It WILL NOT create a new pair with the primer selected
You then have a view of all the primers saved in the database with the following info displayed:
  • ID
  • Name
  • Gene
  • Sequence
  • GC %
  • Tm
  • Chr
  • GRCh37 Coverage --> clickable link to UCSC
  • GRCh38 Coverage --> clickable link to UCSC
  • Amplicon size
  • +/- (strand)
  • Buffer
  • PCR Program
  • Scientist
  • Arrival date
  • Storage Location
  • Last Date Used
  • Status
  • SNP status:
    • Yellow = SNP check has not been run
    • Green = No SNPs detected
    • Red = SNPs detected
    • Blue = SNPs checked by scientist
    • Purple = Gene not recognized in gnomAD
  • Comments
Information on the number of primers and their snp status is displayed. You can click on the numbers to filter according to the snp status.
Clicking on a primer will send you to the editing page of that primer.

Submitting Primers

Submitting Single Primers

The form has the following fields:

  • Primer name
  • Gene name
  • Sequence
  • Buffer
  • PCR Program
  • Arrival date
  • Storage location
  • Comments (optional)
  • Forename
  • Surname
A few checks will be run after you submit a primer:
  • Does the primer name already exist?
  • Does the gene name exist in our "gene file" for mapping?
  • Does the sequence contain characters other that ATGC?
  • Is the primer "In Bank" with a storage location?
Failing one of these checks will result in a failure when submitting the data for a primer.
A mapping process + SNP checking process will be run while submitting to get coverage and possible variants in the primer sequences.

Submitting Pairs of Primers

The form is duplicated for the 2 primers paired up.
In addition to the rest, a coverage is calculated for GRCh37 and GRCh38.

Editing Primers

Editing Single Primers

In editing mode, you can modify the name, buffer, pcr program, arrival date, status, storage location, comments, and name of scientist.
You can also update the last date used using the "Update date used" button.
To delete a primer, you can click the "Delete primer" primer.

If SNPs are detected in a primer, scientists can manually clear them:

  • A table containing the positions of the SNPs on the primer and a link to gnomAD will be present when SNPs are detected.
  • Scientists can them decide for themselves whether the primer is still usable.
  • A button called "SNP Check primer_name" will be present when SNPs are detected.
Clicking this button will change the SNP status in the index table.
